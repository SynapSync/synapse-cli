# synapsync search (deprecated)

:::info Renamed
The `search` command has been replaced by `list --remote`. See the [`list` command](/cli/commands/list) for full documentation.
:::

## Migration

| Old Command | New Command |
|------------|-------------|
| `synapsync search` | `synapsync list --remote` |
| `synapsync search react` | `synapsync list --remote --tag react` |
| `synapsync search --type skill` | `synapsync list --remote --type skill` |
| `synapsync search --category frontend` | `synapsync list --remote --category frontend` |
